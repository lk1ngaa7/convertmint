# CONTENT_RULES — 站点英文内容写作规则

## 1. 基本原则

虽然项目文档是中文，但前台网站文案必须是英文，因为目标用户是英文 Google 搜索用户。

文案风格：

- clear
- practical
- concise
- trustworthy
- not hype-driven
- not keyword-stuffed

不要写成：

```txt
The best and most powerful converter in the world...
```

应该写成：

```txt
Convert kg to lbs instantly. Enter a kilogram value and get the pound result with formula and common examples.
```

---

## 2. 页面文案结构

每个工具页必须包含：

1. H1；
2. subtitle；
3. trust note；
4. result explanation；
5. formula；
6. examples；
7. FAQ；
8. related tools。

---

## 3. H1 规则

H1 必须简短，精准命中主关键词。

推荐：

```txt
Convert kg to lbs
Convert lbs to kg
Military Time Converter
Weight Converter
```

不推荐：

```txt
Free Online Best Accurate Kilograms to Pounds Unit Conversion Calculator Tool
```

---

## 4. Subtitle 规则

Subtitle 解释页面能做什么，不堆关键词。

示例：

```txt
Enter a weight in kilograms and get the pound value instantly, with formula and conversion table.
```

---

## 5. Formula 文案

### kg to lbs

```txt
To convert kilograms to pounds, multiply the kilogram value by 2.20462.
```

公式展示：

```txt
pounds = kilograms × 2.20462
```

### lbs to kg

```txt
To convert pounds to kilograms, divide the pound value by 2.20462.
```

公式展示：

```txt
kilograms = pounds ÷ 2.20462
```

---

## 6. FAQ 写法

FAQ 回答要短，尽量 1-3 句。

示例：

Question：

```txt
How do you convert kg to lbs?
```

Answer：

```txt
Multiply the kilogram value by 2.20462. For example, 80 kg × 2.20462 = 176.37 lbs.
```

---

## 7. 禁止行为

不要：

- 机械重复主关键词；
- 在每段都插入 `convert kg to lbs`；
- 生成无意义长文；
- 编造权威背书；
- 声称医学、营养、法律、金融建议；
- 对体重做价值判断。

尤其注意：重量换算可能和体重相关，文案必须中性。

不要写：

```txt
80 kg is overweight.
```

只写：

```txt
80 kg is equal to 176.37 lbs.
```
